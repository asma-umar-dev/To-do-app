# TaskFlow Pro — Professional Task Manager

**Live Site:** [asma-umar-dev.github.io/To-do-app](https://asma-umar-dev.github.io/To-do-app/)

TaskFlow Pro is a sleek, dashboard-style task management app for organizing daily work, tracking productivity, and staying on top of deadlines — built for individual use with a personalized workspace feel.

## ✨ Features

- **Dashboard Overview** — Personalized greeting, daily quote, and at-a-glance stats:
  - Total Tasks, Completed, Pending, and Completion Rate
- **Task Management**
  - Create tasks with title, description, priority (🔴 High / ⚡ Medium / 🟢 Low), category (💼 Work, 👤 Personal, 📚 Study, 💪 Health), and due date
  - Edit, complete, and delete tasks (with confirmation modal)
  - Recent Tasks preview + full task list view
- **Filtering & Sorting**
  - Filter by All / Pending / Completed / Overdue / High Priority
  - Sort by Newest, Oldest, Priority, Due Date, or Alphabetical
- **Progress Overview** — Visual breakdown of Total, Done, Pending, and Overdue tasks with completion %
- **Analytics**
  - 📊 Tasks by Priority
  - 🗂 Tasks by Category
  - 💡 Daily Productivity Insights
  - 🔥 Activity Heatmap (last 28 days, GitHub-style)
- **Settings**
  - Editable display name / profile
  - 🌙 Dark Mode toggle
  - Custom accent color
  - Reset All Data (danger zone, with confirmation)
- **Keyboard Shortcuts**
  - `Ctrl + N` — New Task
  - `Ctrl + K` — Search
  - `Ctrl + 1` — Dashboard
  - `Ctrl + 2` — My Tasks

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (single-page static app)
- Local data persistence (browser-based storage)
- Hosted via **GitHub Pages**
- Responsive, dashboard-style UI with light/dark theming

## 📁 Getting Started

Clone the repo and open `index.html` directly, or serve it locally:

```bash
git clone https://github.com/asma-umar-dev/To-do-app.git
cd To-do-app
# then open index.html in your browser
# or serve locally:
npx serve .
```
